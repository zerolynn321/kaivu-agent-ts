# Literature Wiki Page Boundaries

Use this reference when page selection is ambiguous.

## Prefer Existing Pages

If an existing page covers the same method, benchmark, claim, finding, formal result, research question, or topic, update that page. Do not create a near-duplicate with a slightly different name.

## Method vs Benchmark

Use `method` for how something is done.

Use `benchmark` for how something is evaluated.

If a paper introduces both a method and a benchmark, create or update both and link them.

## Finding vs Claim

Use `finding` for source-grounded results: measurements, observations, reported outcomes, experiments, or analyses.

Use `claim` for propositions that organize debate and can accumulate supporting, contradicting, or qualifying evidence.

## Formal Result

Use `formal_result` for theorem-like material, including lemmas, corollaries, propositions, conjectures, bounds, guarantees, and proof-dependent claims.

## Topic vs Synthesis

Use `topic` for a field area, subproblem, or research thread.

Use `synthesis` only for a durable integrated view across multiple pages. A topic's normal current picture belongs on the topic page.
